# Professional Portfolio Website Plan (GitHub Pages)

## 🧭 1. Define Goal & Content

### Objective
Create a site that showcases:
- ATE/Test Engineering expertise
- Key projects (NI GP4 migration, PVM module, TestStand work)
- Skills (C#, NI tools, hardware bring-up)
- Contact and resume

### Structure
```
Home
About
Projects
Skills
Experience
Contact
```

---

## 🏗️ 2. Choose Tech Stack

### Recommended
- HTML + CSS + minimal JavaScript
- Optional: Tailwind CSS or Bootstrap

### Future Upgrade
- Jekyll (GitHub-native)
- React + Vite (dynamic UI)

---

## 📁 3. Repository Setup

### Steps
1. Create repository:
```
username.github.io
```

2. Clone repo:
```bash
git clone https://github.com/username/username.github.io.git
```

3. Folder structure:
```
/ 
 ├── index.html
 ├── css/
 ├── js/
 ├── assets/images/
 └── projects/
```

---

## 🎨 4. Design Wireframe

### Home Layout
- Header (Name, Title, Navigation)
- Hero section (Intro)
- Highlights (Projects + Skills)
- Footer (Contact)

### Design Tips
- White background, dark text
- 1 primary + 1 accent color
- Clean fonts (Inter / Roboto)
